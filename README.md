# Mars App

## Purpose of creation
A course project completed as part of a course at the Yandex Academy Lyceum. First experience creating a web application in Flask: user registration, authorization, and adding entries to the work log.

## Technology stack
- Python 3
- Flask
- SQLAlchemy
- SQLite
- WTForms (Flask-WTF)
- Flask-Login

## How to launch
```bash
git clone https://github.com/AndreiDuvakin/MarsApp.git
cd MarsApp
pip install -r requirements.txt
python main.py
```
