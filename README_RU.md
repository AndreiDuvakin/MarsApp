# Mars App

## Цель создания
Учебный проект, выполненный в рамках обучения в Лицее Академии Яндекса. Первый опыт создания веб-приложения на Flask: регистрация пользователей, авторизация, добавление записей в журнал работ.

## Стек технологий
- Python 3
- Flask
- SQLAlchemy
- SQLite
- WTForms (Flask-WTF)
- Flask-Login

## Как запустить
```bash
git clone https://github.com/AndreiDuvakin/MarsApp.git
cd MarsApp
pip install -r requirements.txt
python main.py
```
